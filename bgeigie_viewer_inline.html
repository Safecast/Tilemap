bGeigie Logs
A bGeigie log contains the original data points upon which the Safecast map tiles are based.

This offers additional information and greater resolution, but is limited to a smaller subset of data.

To Add Enter the ID of a log. Click "Search".
(Multiple IDs: use commas)
To Search Select "User ID" or "Text".
You can choose a specific date range,
and status type.
To Clear Click "Remove All" to remove all logs added to the map.
Tip 1/3 To search by date, select "Text" but leave the text field empty.
Tip 2/3 If using Street View, it's recommended to use a larger marker size, such as 32px.
Tip 3/3 The heading line is not accurate in Street View or 45 degree aerial imagery views.
