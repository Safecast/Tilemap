bGeigie Logs
A bGeigie log contains the original data points upon which the Safecast map tiles are based.

This offers additional information and resolution, but is limited to a smaller subset of data.

To Add Enter the ID of a log. Click "Search".
(Multiple IDs: use commas)
To Search Select "User ID" or "Text".
You can choose a specific date range,
and status type.
To Clear Click "Remove All" to remove all logs added to the map.
Tip 1/5 To search by date, select "Text" but leave the text field empty.
Tip 2/5 To not include aircraft flights, set the status type to "Approved."
Tip 3/5 If using Street View, it's recommended to use a larger marker size, such as 32px.
Tip 4/5 For best performance when displaying many logs, disable the bearing line on the marker.
Tip 5/5 The bearing line is not accurate in Street View or 45 degree aerial imagery views.
